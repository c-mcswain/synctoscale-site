Pre-Start Checklist

Automatically spin up a pre-start checklist for every new hire. Make sure hardware, access, paperwork, and welcome moments are ready before day one — with Nova highlighting gaps in real time.

Overview

The Pre-Start Checklist template ensures critical onboarding steps are completed before a new hire’s first day. When a new employee is created in your HRIS (or within a defined pre-start window), SyncToScale automatically creates a checklist of tasks for People Ops, IT, and the hiring manager — with Nova flagging missing details that could delay onboarding.

How it works

  1. A new hire is created in the HRIS with a start date.
  2. SyncToScale builds a structured onboarding profile.
  3. Nova checks for missing key fields and potential blockers.
  4. A pre-start checklist is created in your task/project tool.
  5. Tasks are assigned with due dates relative to the start date.
  6. Stakeholders receive a Slack or email summary.

Workflow steps

  1. Trigger: New hire created or pre-start window reached.
  2. Normalize data: HRIS fields mapped to onboarding profile.
  3. Nova validation: Missing start date, manager, location, hardware.
  4. Create checklist: Based on department and role.
  5. Assign tasks: With relative due dates.
  6. Notify: Slack or email summary sent.

Field mappings

  • firstName + lastName → Checklist title
  • hireDate → Due date baseline
  • department → Checklist variant
  • managerEmail → Task ownership
  • employmentType → Hardware rules

Nova insights

  • Flags missing hardware requests.
  • Warns when pre-start window is too short.
  • Highlights hires at risk of first-day delays.

Set up this template

  1. Select Pre-Start Checklist in Templates.
  2. Connect HRIS and task/project tools.
  3. Define pre-start timing rules.
  4. Enable Nova validations.
  5. Test, then activate.

Template summary

  • Trigger: New hire / pre-start window
  • Runtime: Immediate
  • Owner: People Ops · IT · Manager
  • Systems: HRIS · Task tool · Slack/Email

Best practices

  • Use a 7–10 day pre-start window.
  • Keep checklist focused and role-aware.
  • Review Nova warnings daily.

Nova will help you…

  • Prevent first-day surprises.
  • Identify late or missing prep tasks.
  • Continuously improve onboarding readiness.
