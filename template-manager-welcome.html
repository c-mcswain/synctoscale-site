Overview

This workflow ensures managers receive a consistent, actionable onboarding packet as soon as a new hire is created. It reduces last-minute scrambling and improves early employee experience.

How it works

  1. New hire is created in the HRIS
  2. Manager and role data are enriched
  3. Nova validates onboarding readiness
  4. A Manager Welcome Packet is generated
  5. The packet is delivered via Slack or email

What managers receive

  • New hire overview â€” role, start date, team
  • First-week priorities
  • 30 / 60 / 90-day guidance
  • Key meetings & milestones
  • Checklist & onboarding resources

Set up this workflow

  1. Connect HRIS and Slack or Email systems
  2. Review the default packet structure
  3. Customize checklist items if needed
  4. Enable Nova validations
  5. Activate the workflow

Workflow summary

  • Trigger: New hire created
  • Owner: People Ops
  • Output: Manager Welcome Packet
