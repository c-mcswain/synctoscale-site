Automatically send hiring managers a structured welcome packet when a new hire is created in the HRIS. Includes actionable checklists, key dates, onboarding expectations, and Nova insights to prepare them before day one.

Overview

The Manager Welcome Packet template ensures hiring managers have everything they need to support a new employee's first days. Triggered when a new hire is created in the HRIS, SyncToScale generates a personalized packet including required tasks, important dates, onboarding expectations, and Nova-driven insights.

How it works

  1. New hire is created in the HRIS.
  2. Profile is enriched with role, team, and manager data.
  3. Nova checks for missing onboarding prerequisites.
  4. A personalized packet is generated.
  5. Delivered via Slack or email.
  6. Logged for People Ops visibility.

Set up this template

  1. Select Manager Welcome Packet from Templates.
  2. Connect HRIS and Slack/Email systems.
  3. Customize checklist items.
  4. Enable Nova validations.
  5. Activate the workflow.
