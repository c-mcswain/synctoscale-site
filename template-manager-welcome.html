Workflow Template

Manager Welcome Packet

Automatically equip hiring managers with a clear, structured welcome packet when a new hire is created — including key dates, onboarding expectations, and Nova-powered insights before day one.

Onboarding Beginner–Intermediate New Hire Trigger HRIS · Slack · Email

Overview

The Manager Welcome Packet template ensures managers are prepared and aligned before a new employee’s first day. When a new hire is created in the HRIS, SyncToScale generates a personalized packet with required tasks, expectations, and Nova-driven insights.

How it works

  1. New hire is created in the HRIS.
  2. Role, team, and manager data is enriched.
  3. Nova checks onboarding readiness and dependencies.
  4. A personalized manager packet is generated.
  5. Packet is delivered via Slack or email.
  6. Activity is logged for People Ops visibility.

Set up this template

  1. Select Manager Welcome Packet from Templates.
  2. Connect HRIS and Slack or Email systems.
  3. Customize checklist and guidance content.
  4. Enable Nova validations.
  5. Activate the workflow.

Template summary

  • Trigger: New hire created
  • Runtime: Immediate
  • Owner: People Ops
  • Systems: HRIS · Slack · Email

Best practice defaults

  • Set expectations before day one.
  • Include 30 / 60 / 90 guidance.
  • Standardize manager readiness.

Nova will help you…

  • Flag onboarding readiness gaps.
  • Predict manager workload bottlenecks.
  • Create consistent onboarding experiences.
