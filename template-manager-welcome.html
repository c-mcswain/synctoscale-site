Manager Welcome Packet

Automatically generate a structured welcome packet for hiring managers when a new hire is created — setting expectations, surfacing key dates, and preparing managers before day one.

Onboarding Beginner–Intermediate New Hire Trigger HRIS · Slack · Email

Overview

This workflow generates a Manager Welcome Packet automatically when a new hire is created in the HRIS. The packet helps managers prepare ahead of time by outlining responsibilities, timelines, and onboarding expectations.

How it works

  1. New hire is created in the HRIS.
  2. Manager and role data are enriched.
  3. Nova checks onboarding readiness.
  4. A personalized Manager Welcome Packet is generated.
  5. Packet is delivered via Slack or email.

Set up this workflow

  1. Connect HRIS and Slack or Email systems.
  2. Review and customize the default packet structure below.
  3. Enable Nova validations.
  4. Activate the workflow.

Default Manager Welcome Packet

What managers receive

  • New hire overview (role, start date, team)
  • First-week expectations and priorities
  • 30 / 60 / 90-day guidance
  • Key meetings and milestones
  • Onboarding checklist and resources
