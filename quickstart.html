Quickstart

Get your first SyncToScale workflow running in minutes. This guide shows authentication, triggering a run, and checking results.

What you’ll need

  • A SyncToScale workspace
  • An API key
  • A workflow ID

Authentication

Send your API key as a bearer token on every request.

Authorization: Bearer YOUR_API_KEY

POST Run a workflow

curl -X POST "https://api.synctoscale.me/v1/workflows/run" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "workflow_id": "wf_12345",
  "payload": {
    "event": "new_hire_created",
    "employee_id": "E-1042"
  }
}'

GET Check run status

curl -X GET "https://api.synctoscale.me/v1/runs/run_abc123" \
-H "Authorization: Bearer YOUR_API_KEY"
