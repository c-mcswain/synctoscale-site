Manager Welcome Packet

Template

Automatically send hiring managers a structured welcome packet when a new hire is created in the HRIS. Includes actionable checklists, key dates, onboarding expectations, and Nova insights to prepare them before day one.

Category: Employee Onboarding Difficulty: Beginner Trigger: New hire created Systems: HRIS · Email · Slack

Overview

The Manager Welcome Packet template ensures hiring managers have everything they need to support a new employee's first days. Triggered when a new hire is created in the HRIS, SyncToScale generates a personalized packet including required tasks, important dates, onboarding expectations, and Nova-driven insights (e.g., “Your new hire has no laptop request yet”).

When to use this template

  • Your managers often feel unprepared on day one.
  • Critical pre-start steps (equipment, access, introductions) are missed.
  • You want consistent onboarding across teams.
  • You want Nova to surface missing data early.

What this template automates

  • Sends the manager a welcome packet via email or Slack.
  • Provides key dates (hire date, first day, orientation schedule).
  • Includes a pre-start checklist customized by role.
  • Surfaces Nova insights about missing or inconsistent onboarding data.
  • Links directly to IT & People Ops onboarding workflows.

How it works

  1. A new hire is created in the HRIS.
  2. SyncToScale enriches the profile with role, department, location, and manager info.
  3. Nova checks for common onboarding issues (missing hardware, no start date, etc.).
  4. A personalized Manager Welcome Packet is generated.
  5. Packet is delivered through Slack DM or email.
  6. A summary record is logged for People Ops visibility.

Workflow steps

  1. Step 1 — Trigger: New hire created

    Runs when an HRIS event such as employee.created or employmentStatus = "Hired" is detected.

  2. Step 2 — Build onboarding profile

    SyncToScale constructs a structured profile that includes:

    • Full name & preferred name
    • Start date
    • Role & team
    • Manager information
    • Location & timezone

  3. Step 3 — Nova validation

    Nova checks for issues and missing onboarding prerequisites:

    • Missing hardware request
    • Missing start date
    • Manager has multiple new hires on the same day
    • Department mismatch between HRIS & directory

  4. Step 4 — Generate welcome packet

    SyncToScale composes a structured packet customized for the manager and the new hire.

  5. Step 5 — Deliver via Slack or Email

    The packet is delivered to the manager through their preferred channel with all key links.

  6. Step 6 — Log completion

    A record is sent to People Ops so they can track onboarding preparation.

Field mappings

HRIS → Manager Packet

{
  "manager": "Sarah Miles",
  "newHire": "Ava Lopez",
  "startDate": "May 1, 2025",
  "team": "People",
  "role": "HR Coordinator",

  ]
}

Error handling

  • If the manager is not found in the directory, fallback to People Ops.
  • Missing hire date triggers a Nova warning and blocks packet generation.
  • Slack DM failures retry 3 times then fallback to email.

Nova insights for this template

  • Detects missing onboarding prerequisites.
  • Highlights upcoming workload for managers.
  • Surfaces inconsistencies between HRIS and identity data.
  • Predicts which onboarding tasks may be delayed.

Set up this template

  1. Select Manager Welcome Packet in your Templates Library.
  2. Connect your HRIS and Slack/Email systems.
  3. Customize your checklist items.
  4. Enable Nova onboarding validations.
  5. Activate the workflow.
