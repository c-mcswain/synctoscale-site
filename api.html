API Reference

REST API endpoints to power your integrations.

Authentication

SyncToScale uses Bearer token authentication for all API requests.


Authorization: Bearer <YOUR_API_KEY>

Endpoints

List Users


GET /v1/users

Create Workflow


POST /v1/workflows
{
  "name": "Sync Salesforce Leads",
  "trigger": "lead.created",
  "actions": [...]
}

Execute Workflow


POST /v1/workflows/{id}/run
{
  "payload": {...}
}

Rate Limits

Default rate limits:

  • 10,000 requests / hour
  • Burst capacity: 200 requests / second
  • Webhook retries: exponential backoff
