Overview

This workflow runs automatically before a new hire‚Äôs start date to confirm that access, equipment, and onboarding logistics are complete. It reduces day-one friction and gives teams confidence that nothing critical was missed.

How it works

  1. New hire is created in the HRIS
  2. Start date determines checklist timing
  3. Tasks are assigned to IT, People Ops, and managers
  4. Status is tracked centrally
  5. Nova flags incomplete items before day one

Default pre-start checklist

  • Employee account provisioning (email, Slack, core tools)
  • Laptop and equipment prepared and shipped
  • Payroll and tax forms completed
  • Benefits enrollment instructions sent
  • First-week meetings scheduled
  • Manager welcome packet delivered

Set up this workflow

  1. Connect HRIS and IT systems
  2. Define lead time (e.g., 5 business days before start)
  3. Assign checklist owners by task type
  4. Enable Nova readiness checks
  5. Activate the workflow

Workflow summary

  • Trigger: New hire + start date
  • Owner: People Ops ¬∑ IT
  • Output: Task completion status
