Developer-friendly API & webhooks.

Extend SyncToScale, build custom automations, or plug the platform into your own internal tools with a simple REST API.

Core endpoints

GET   /workflows
POST  /workflows
GET   /integrations
POST  /connections
GET   /events
GET   /errors
POST  /webhooks/validate

Sample request

POST /connections
Content-Type: application/json

{
  "type": "salesforce",
  "auth": {
    "client_id": "xxxx",
    "client_secret": "xxxx",
    "refresh_token": "xxxx"
  }
}

In a real implementation, you’d store secrets securely and rotate them regularly. For this portfolio project, you can keep this API purely illustrative.
