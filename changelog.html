Changelog

A running record of notable changes, improvements, and fixes made to the SyncToScale platform.

January 2025
v0.9.0

Added

  • Integration detail pages for HubSpot, Salesforce, Notion, Asana, and Google Workspace.
  • Dedicated Webhooks integration documentation outlining validation and retry patterns.
  • Expanded Security page covering access control, incident response, and shared responsibility.

Improved

  • Standardized integration page layout to align with Docs / Quickstart patterns.
  • Clarified troubleshooting guidance across integration detail pages.
  • Navigation consistency across Resources, Docs, and Integrations.
December 2024
v0.8.2

Added

  • Initial Integrations library with Slack, Jira, BambooHR, and Gusto.
  • Workflow Templates library with printable, recruiter-friendly artifacts.

Improved

  • Refined Resources page structure to separate Docs, Templates, and Integrations.
  • Improved Nova Botpress widget styling and placement.

Fixed

  • Navigation overlap issues on mobile viewports.
  • Inconsistent spacing across template detail pages.
November 2024
v0.7.0

Added

  • Initial SyncToScale site structure and design system.
  • Core Docs sections covering authentication, triggers, actions, and mappings.

Improved

  • Base styles.css tokens for a consistent light SaaS UI.
  • Global navigation and layout responsiveness.
